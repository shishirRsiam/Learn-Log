## **Step 1: Generate CSR (Certificate Signing Request)**

Since we’re **not using Name.com hosting**, we must generate a CSR manually.

### **1.1 Generate Private Key and CSR**

```bash
# Navigate to a safe folder
mkdir -p ~/ssl && cd ~/ssl

# Generate private key
openssl genrsa -out shishirrsiam.dev.key 2048

# Generate CSR (will prompt for domain and contact info)
openssl req -new -key shishirrsiam.dev.key -out shishirrsiam.dev.csr
```

**CSR Details Example:**

| Field               | Example                    |
| ------------------- | -------------------------- |
| Common Name         | `www.shishirrsiam.dev`     |
| Organization        | `Gemten`                   |
| Organizational Unit | `SWE`                      |
| City                | `Lalmonirhat`              |
| State               | `Lalmonirhat`              |
| Country             | `BD`                       |
| Email               | `shishir.siam01@gmail.com` |

> Keep the private key safe — you’ll need it when installing SSL.

## **Step 7: Nginx Reverse Proxy Setup (Manual)**

* Goal: Serve `http://62.72.46.17:8072/web` via `https://shishirrsiam.dev`

1. **Create Nginx config:**

```bash
sudo nano /etc/nginx/sites-available/shishirrsiam.dev
```

Paste:

```nginx
server {
    listen 443 ssl;
    server_name shishirrsiam.dev www.shishirrsiam.dev;

    ssl_certificate /etc/ssl/shishirrsiam.dev/certificate.crt;
    ssl_certificate_key /etc/ssl/shishirrsiam.dev/shishirrsiam.dev.key;
    ssl_trusted_certificate /etc/ssl/shishirrsiam.dev/ca_bundle.crt;

    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    location / {
        proxy_pass http://62.72.46.17:8072/web/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_redirect /web/ /;
    }
}

server {
    listen 80;
    server_name shishirrsiam.dev www.shishirrsiam.dev;
    return 301 https://$host$request_uri;
}
```

2. **Enable site & reload Nginx:**

```bash
sudo ln -sf /etc/nginx/sites-available/shishirrsiam.dev /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

3. **Test site:**

```
https://shishirrsiam.dev
https://www.shishirrsiam.dev
```

## **Step 8: Full Automation Script**

Save as `/root/auto-ssl-reverse-proxy-public-ip.sh`:

```bash
#!/bin/bash
set -e

DOMAIN="shishirrsiam.dev"
SSL_DIR="/etc/ssl/$DOMAIN"
APP_URL="http://62.72.46.17:8072/web"
LOCAL_SSL_FOLDER="/root/ssl_source"

# Check SSL files
for file in certificate.crt shishirrsiam.dev.key ca_bundle.crt; do
    if [ ! -f "$LOCAL_SSL_FOLDER/$file" ]; then
        echo "ERROR: $file not found"
        exit 1
    fi
done

# Install Nginx
sudo apt update
sudo apt install -y nginx

# Copy SSL files
sudo mkdir -p "$SSL_DIR"
sudo cp "$LOCAL_SSL_FOLDER/"*.crt "$SSL_DIR/"
sudo cp "$LOCAL_SSL_FOLDER/shishirrsiam.dev.key" "$SSL_DIR/"
sudo chmod 600 "$SSL_DIR/"*.key
sudo chmod 644 "$SSL_DIR/"*.crt

# Nginx config
NGINX_CONF="/etc/nginx/sites-available/$DOMAIN"
sudo tee "$NGINX_CONF" > /dev/null <<EOL
server {
    listen 443 ssl;
    server_name $DOMAIN www.$DOMAIN;

    ssl_certificate $SSL_DIR/certificate.crt;
    ssl_certificate_key $SSL_DIR/shishirrsiam.dev.key;
    ssl_trusted_certificate $SSL_DIR/ca_bundle.crt;

    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;

    location / {
        proxy_pass $APP_URL/;
        proxy_set_header Host \$host;
        proxy_set_header X-Real-IP \$remote_addr;
        proxy_set_header X-Forwarded-For \$proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto \$scheme;
        proxy_redirect /web/ /;
    }
}

server {
    listen 80;
    server_name $DOMAIN www.$DOMAIN;
    return 301 https://\$host\$request_uri;
}
EOL

# Enable & reload
sudo ln -sf "$NGINX_CONF" /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

echo "✅ SSL + Reverse Proxy setup complete."
```
